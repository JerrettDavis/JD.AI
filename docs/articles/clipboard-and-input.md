# Clipboard and Input

## Interactive input
- Ghost-text tab completions for slash commands
- Command history (up/down arrows)
- Multi-line input support

## Clipboard paste support
When pasting content, JD.AI detects and organizes it:
- **Text blocks**: `[Pasted content #1 1000 chars]`
- **Image data**: `[Pasted Image #1 100kb]`
- **File content**: `[Pasted PDF File #1 2.5kb]`

Pasted content appears as attachments that can be removed individually before sending.

## History browser
Press ESC twice to open the interactive history browser:
- Browse previous turns
- Rollback to any previous point in the conversation
- View token usage per turn

## Input tips
- Use tab for slash command completion
- Up/down arrows for command history
- Paste large code blocks — they're automatically detected and chunked
- Long inputs wrap properly within the terminal width
